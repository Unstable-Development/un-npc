======================================
PRELUDE NPC - ADMIN COMMAND REFERENCE
======================================

🎯 RECOMMENDED: USE THE CONTROL PANEL
--------------------------------------
/npcpanel                   - Open tablet-style visual control panel (Admin)
                              All commands below can be done through the UI!

🆕 NEW: IN-GAME BUILDERS
--------------------------
/createzone                 - Start in-game zone builder (Admin)
                              Create guard zones visually with markers
/createroute                - Start in-game route builder (Admin)
                              Create patrol routes by driving/walking

HUNTER COMMANDS (Everyone)
--------------------------
/chaseme [level] [count]    - Spawn hunters (level:1-5, count:1-10)
/chaseoff                   - Remove all hunters

GUARD ZONE COMMANDS (Admin)
----------------------------
/spawnguards [zone]         - Spawn guards in zone
/clearguards [zone/all]     - Clear guards from zone
/listzones                  - List all guard zones
/toggleguardzones           - Toggle guard zones on/off

PATROL COMMANDS (Admin)
-----------------------
/spawnpatrol [route]        - Spawn patrol on route
/clearpatrols               - Clear all patrols
/listpatrols                - List all patrol routes

BODYGUARD COMMANDS (Admin)
--------------------------
/givebodyguard [id] [tier]  - Give bodyguard to player (tier:1-3)
/clearbodyguards [id]       - Remove bodyguards from player
